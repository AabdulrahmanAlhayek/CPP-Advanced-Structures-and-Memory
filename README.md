# Advanced Structures & Memory Management Notebook (C++) 🧠💻

مستودع تقني مخصص لتطبيق المفاهيم المتقدمة في تنظيم هيكلية الأكواد (Code Structuring) وإدارة الكائنات داخل الميموري باستخدام لغة ++C.

## 📌 المحتويات البرمجية للمستودع (Table of Contents)

### 1. التغليف الداخلي للهياكل (Nested Struct Inside Class)
- **الملف:** `01_Struct_Inside_Class.cpp`
- **الشرح:** تطبيق مبدأ الـ Encapsulation الكامل عبر إخفاء الـ Struct ونُسخ البيانات داخل قسم الـ `private` لمنع الوصول المباشر من الـ `main` وإتاحتها فقط عبر دالات القراءة الرسمية.

### 2. تداخل الكائنات والتركيب ديناميكياً (Nested Class & Composition)
- **الملف:** `02_Class_Inside_Class_Composition.cpp`
- **الشرح:** بناء كلاس عنوان ممتلك بالكامل داخل كلاس الشخص (Has-A Relationship)، وتمرير البيانات بشكل مرن وديناميكي عبر الـ Member Initialization List دون تثبيت قيم جامدة.

### 3. تمرير الكائن الحالي عبر دالات الـ Static
- **الملف:** `03_This_Pointer_And_Static_Function.cpp`
- **الشرح:** تجربة برمجية متقدمة لحل قيد عمى الدالة الساكنة عن الكائن الحالي؛ حيث يتم استخدام الـ `*this` لحزم بيانات الغرض الحالي بالكامل وتمريرها كبارامتر لطابعة الـ `static`.

### 4. المصفوفات الديناميكية للكائنات (Vector of Objects)
- **الملف:** `04_Vector_Of_Objects.cpp`
- **الشرح:** تطبيق عملي يوضح كيفية بناء كائنات مجهولة الهوية ديناميكياً وحقنها مباشرة داخل الـ `std::vector` باستخدام حلقات التكرار وكفاءة حجز الذاكرة.
